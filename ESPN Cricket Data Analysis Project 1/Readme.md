## Project Overview

The ESPN Cricket Analytics Dashboard is a Power BI project that analyzes cricket player performance using ESPN cricket data. The dashboard provides insights into batting, bowling, and fielding statistics, helping users understand player performance through interactive visualizations and key performance indicators.

## Objectives

* Analyze player batting performance.
* Analyze player bowling performance.
* Analyze player fielding performance.
* Create interactive dashboards for cricket analytics.
* Generate meaningful insights from cricket statistics.

## Tools Used

* Power BI
* Power Query
* Microsoft Excel
* Web Scraping
* DAX

## Dataset

The dataset was collected from ESPN Cricket Statistics through web scraping and contains detailed player statistics related to batting, bowling, and fielding performance.

## Project Workflow

### 1. Data Collection

* Collected cricket statistics from ESPN using web scraping techniques.

### 2. Data Cleaning

* Removed inconsistencies and unnecessary data.
* Standardized column names and data formats.
* Handled missing and duplicate values.

### 3. Data Modeling

* Defined relationships between datasets.
* Created calculated columns and measures using DAX.

### 4. Data Analysis

* Performed batting performance analysis.
* Performed bowling performance analysis.
* Performed fielding performance analysis.

### 5. Dashboard Development

Created multiple interactive dashboard pages including:

* Overview Dashboard
* Batting Analysis
* Bowling Analysis
