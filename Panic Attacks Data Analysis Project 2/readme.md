# 🧠 Panic Attacks Data Analysis

## 📌 Project Overview

The **Panic Attacks Data Analysis** project is an end-to-end healthcare analytics solution that explores the factors contributing to panic attacks using modern data analytics tools. The project focuses on cleaning, transforming, analyzing, and visualizing patient data to uncover meaningful insights about symptoms, lifestyle factors, and demographic patterns.

This project demonstrates the complete data analytics workflow—from data preparation in **Snowflake** using **SQL** to building an interactive dashboard in **Power BI**.

---

## 🚀 Objectives

* Analyze panic attack patterns across different age groups and genders.
* Identify the most common symptoms experienced during panic attacks.
* Study the relationship between sleep, lifestyle, and panic attack severity.
* Build an interactive dashboard for exploring patient data through dynamic filters.
* Generate actionable insights using visual analytics.

* Medical History Analysis

  * Trigger
  * Medical History

## 🔍 Key Insights

* Identified the most frequently reported symptoms experienced during panic attacks.
* Compared panic attack frequency across different age groups.
* Analyzed average sleep hours and their relationship with panic scores.
* Explored panic attack duration among patients.
* Examined weekly alcohol consumption patterns.
* Investigated trigger factors responsible for panic attacks.
* Compared panic attack severity based on gender.
* Studied the impact of medical history on panic attack frequency.
